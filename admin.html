Secure Payment Gateway - Privacy-First System

üìù Important Note

No admin dashboard needed! This system uses temporary storage for maximum privacy. All transaction data is automatically deleted after completion (60 seconds) or failure (5 minutes). There is no persistent data to monitor.

How This System Works

Step 1: User Payment
User sends full amount to your escrow wallet (one transaction only)

Step 2: Blockchain Verification
System waits for 3 confirmations (~45-90 seconds on Sepolia)

Step 3: Automatic Forwarding
System forwards amount to destination minus 1% fee

Step 4: Data Deletion
All transaction records automatically deleted after 60 seconds

üõ°Ô∏è Privacy & Security Features

  • Single Transaction: User only sends to your wallet
  • Hidden Identity: Destination never sees sender's address
  • Automatic Fee: 1% deducted when forwarding to destination
  • Temporary Storage: All data in memory, not saved to disk
  • Auto-Deletion: Completed transactions deleted after 60 seconds
  • Failed Cleanup: Failed transactions deleted after 5 minutes
  • No Database: Nothing persists between deployments
  • Serverless: Runs entirely on Vercel (free tier)

Monitoring Your System

Check Blockchain Directly:
Monitor your escrow wallet on Etherscan:
- Sepolia Testnet
- Ethereum Mainnet

View Vercel Logs:
1. Go to Vercel dashboard
2. Click your project
3. Click "Functions" tab
4. See real-time logs of verification and forwarding

Check Your Wallet Balance:
Your escrow wallet accumulates 1% fees from each transaction. Monitor it directly in MetaMask or on blockchain explorers.

Configuration

Your Escrow Wallet:
Update in index.html:
ESCROW_WALLET: '0xYourWalletAddress'

Environment Variables (Vercel):
- RPC_URL: Your Alchemy API URL
- ADMIN_PRIVATE_KEY: Your escrow wallet private key

Fee Percentage:
Default is 1%. To change, update in both:
- index.html: FEE_PERCENTAGE: 1
- api/verify.js: FEE_PERCENTAGE: 1
